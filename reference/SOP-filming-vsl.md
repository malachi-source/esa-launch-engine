# SOP for Filming VSL

**Instructions:**

  - Video should be 3-5 minutes in length
  - Should be filmed in a professional studio setting
  - Video's should be shot in Youtube format (Horizontal)
  - Speak directly to ONE person, not the whole audience
  - Record each video in ONE take if possible, if not try to do it in 2 takes (Body + CTA)
  - Be VERY direct with the BOLD statements and HOOKS
  - Feel the emotion in every word you speak (the more they feel like you understand them, the better results we'll get)
  - Change any words to fit your personality and tone, it doesn't have to be word for word.
  - Record it in a professional studio setting and use a teleprompter
      - Or set your phone up on a tripod and use a teleprompter app to read the lines
      - Also, you can record in different clips and if we need to edit them together, we can do that (Just keep the camera in the same place)
