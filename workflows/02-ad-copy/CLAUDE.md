# 02 Ad Copy — working notes

## What this step does
Generates short-form Meta ad copy (text, not video scripts). Two prompt variants live here:
- `PROMPT-ad-copy.md` — the general "ULTIMATE META AD COPY PROMPT (MASTER VERSION)": 10 unique short-form copy variations, 3–5 sentences, under 200 words each, structure Pain → Connection → Transformation → CTA.
- `PROMPT-event-ticket-ad.md` — the specialized "ULTIMATE EVENT TICKET SALES AD PROMPT": 10 variations, 3–5 sentences, under 160 words, structure Hook → Agitate → Reframe → Transform → CTA, tuned for selling tickets to high-ticket business events.

## Inputs it needs (from the onboarding form / earlier steps)
General ad copy prompt:
- Audience profile (demographics, pain points, desires, objections, identity labels)
- Offer/event details (name, date, location, CTA, transformation, pricing, benefits)

Event ticket prompt (more detailed):
- Event details (name, date/location, ticket tiers/price, format, capacity, core promise, benefits, social proof)
- Target audience profile (demographics + psychographics: pain, desires, objections, verbatim language)
- Strategic context (awareness stage, USP, urgency elements, traffic temperature)

## Output it produces
10 Meta-compliant ad copy variations per campaign, each unique in hook/style/angle, approval-ready. The event ticket prompt also outputs strategic notes, headline options, and CTA button text per ad.
