# 12-Webinar Slides — working notes

## What this step does
Generates a complete done-for-you webinar slide deck (slide-by-slide: headline + script + design note)
that teaches and converts attendees into the client's high-ticket offer. Part of ESA's fulfillment
("we write all the slides for them, completely DFY"). Runs after the foundation + offer are known.

## Inputs it needs
- Event/offer details + the high-ticket offer & price
- The Target Audience Profile (from 00-foundation)
- The big idea / core promise
- Transformation + proof
- The 3 false beliefs to break (vehicle / internal / external)

## Output
A full webinar deck outline (Perfect-Webinar style): big promise → 3 secrets → stack → price →
bonuses → guarantee → scarcity → CTA → objection slides.
