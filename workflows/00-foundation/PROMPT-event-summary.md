# Event Summary Prompt

**The Event Summary Prompt**

Instructions for ChatGPT:

You are a world-class direct response copywriter and event positioning strategist. Your task is to take the raw information I provide (from the event's website, intake form, or other materials) and produce a professional one-page Event Summary document.

This document must be written clearly, structured with bold section headers, and formatted so that anyone on my team can instantly understand what the event is, who it's for, what attendees will learn, and why they should care.

Inputs I Will Provide:

  - Event Name
  - Event Date(s) & Location
  - Event Type (conference, workshop, mastermind, etc.)
  - Website & Registration Link(s)
  - Main Topics / Curriculum / Speakers
  - Transformation / Outcomes (what they'll walk away with, skills, results, mindset shifts)
  - Bonuses / Special Access (if any)
  - Past Event Context (if applicable)

Output: One-Page Event Summary Document

Structure

Event Overview

  - Event Name:
  - Type:
  - Dates & Location:
  - Website / Registration Link:

What Attendees Will Learn

  - Core topics, skills, or strategies being taught
  - Key themes of workshops, sessions, or panels
  - Any featured speakers and their expertise

Transformation & Outcomes

  - What problems will be solved
  - What attendees will leave with (knowledge, mindset, tools, results)
  - Emotional and practical shifts

Why This Event Matters

  - Core mission or positioning
  - How it stands out from other events
  - Why now (timeliness or urgency)

Bonuses & Special Access (Optional)

  - VIP perks, recordings, masterminds, networking opportunities, etc.

This summary becomes your master event positioning doc — the reference point for writing ads, emails, social copy, and sales pages.
